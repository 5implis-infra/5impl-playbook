## Quick Reference

The 5impl.is system consists of **41 AI agents** organized in 7 departments, all running on Paperclip:

| Department | Agents | Key flows |
|---|---|---|
| Executive | CEO, GovernanceAuditor | Orchestration, auditing |
| Editorial | 15 agents | Content pipeline → publish |
| Sales | 8 agents | Lead → contract → provisioning |
| SaaS Church | 7 agents | Subscription lifecycle |
| Support | 3 agents | 24/7 WhatsApp RAG + incidents |
| Financial | 5 agents | Token quota + billing + reporting |

**Tech stack:** Paperclip (agents) · Directus (CMS/data) · n8n (workflows) · LiteLLM (LLM proxy) · Hermes (email) · Astro (site)
