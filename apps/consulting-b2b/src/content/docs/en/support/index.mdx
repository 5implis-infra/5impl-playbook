# Support

5implis offers support through multiple channels. Use the right channel for the type of situation — this ensures the fastest possible response.

| Channel | When to use | Response time |
|---------|-------------|---------------|
| **WhatsApp** | Quick questions, confirmations, daily updates | Up to 2h during business hours |
| **Email** | Formal requests, contracts, invoices, scope changes | Up to 24 business hours |
| **Paperclip** | Project issues, bugs, feature requests (consulting) | Per project SLA |
| **Telegram** | Critical incidents — use only if the system is down | Immediate (P1 alert) |
