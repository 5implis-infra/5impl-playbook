# Agentes

A 5implis opera com agentes de IA especializados — cada um com uma função clara, trabalhando de forma coordenada. Abaixo, um mapa de todos os agentes organizados pela área do negócio que atendem.

---

## Como os agentes trabalham

Os agentes se comunicam entre si por eventos. Quando um agente conclui uma tarefa, ele notifica o próximo agente do processo — como uma esteira de produção onde cada pessoa passa o trabalho para a próxima de forma automática.

Você entra nessa esteira apenas quando uma decisão humana é necessária: aprovar uma proposta, validar uma configuração, confirmar uma entrega.
