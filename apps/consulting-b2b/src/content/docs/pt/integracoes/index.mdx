A plataforma 5implis se conecta às ferramentas que você já usa e às que fazem parte da sua operação. Abaixo, uma visão geral de cada integração e para que ela é usada.

---

## Ferramentas conectadas

| Ferramenta                | Para que serve no seu negócio                                                       |
| ------------------------- | ----------------------------------------------------------------------------------- |
| **WhatsApp (Zernio)**     | Comunicação com leads e clientes — mensagens automáticas e atendimento              |
| **Gateway de Pagamento**  | Cobranças recorrentes, confirmação de pagamentos, gestão de inadimplência           |
| **Assinatura Digital**    | Envio e coleta de assinatura em contratos e propostas                               |
| **Cloudflare**            | Publicação automática do seu site após criação de novos conteúdos                   |
| **Canva**                 | Geração de imagens para posts e newsletter a partir dos seus templates de marca     |
| **Google Search Console** | Monitoramento de posicionamento nos buscadores para orientar a produção de conteúdo |
| **Telegram**              | Alertas internos para você sobre eventos críticos da operação                       |
| **Coolify**               | Infraestrutura dos ambientes da Church Platform (provisionamento automático)        |
| **LiteLLM**               | Gerenciamento das cotas de uso de IA por cliente                                    |

---

## Como as integrações funcionam

Cada ferramenta é conectada uma vez, durante o setup do seu projeto, e depois opera de forma automática. Você não precisa entrar no painel de nenhuma dessas ferramentas para que a plataforma funcione — os agentes cuidam de tudo.

Se uma integração precisar de ajuste ou atualização de credenciais, a equipe 5implis cuida do processo.
